PerfectJobs

Version1.0
PlatformHytale Server Plugin
LanguagesEN, FR, ES, DE, IT, PT
DependenciesNone (PerfectEconomy optional for money rewards)

Complete job and profession system with 4 professions, 100 levels, evolving tools, specializations, daily quests, leaderboards, HUD tracker, and a full admin panel.

Installation

  1. Drop PerfectJobs-1.0.jar into your server's mods/ folder
  2. Start or restart the server
  3. Config files are auto-generated on first run
  4. Players can start working immediately by mining, chopping, hunting, or farming

Features

  • 4 professions: Miner, Lumberjack, Hunter, Farmer
  • 100 levels per job with exponential XP curve
  • Evolving tools in 4 tiers, awarded based on job level
  • Specializations unlocked at level 25 with +50% XP bonus
  • Daily quests (5 per job) that scale with player level
  • HUD tracker showing XP progress for all 4 jobs
  • Leaderboards per job and global (total level)
  • Admin UI with quest templates, XP sources, player editor, and XP boost
  • PerfectEconomy integration for money rewards on quest completion
  • Independent progress in all 4 jobs simultaneously
  • Auto-save every 5 minutes

Admin Panel

Access with /jobs admin. Contains 5 tabs:

  • Quest Templates - Create, edit, and delete custom quest templates per job
  • Players - Browse players, edit levels/XP, reset progress
  • XP Sources - View, add, adjust, or remove XP sources per job
  • XP Boost - Set global XP multiplier (0.1x to 10x, quick buttons: 1x/2x/3x/5x)
  • Global Stats - Total players, total XP earned, average level, per-job breakdowns

API & Placeholders

PerfectJobs provides a static API for other plugins:

JobAPI.getLevel(uuid, "miner")         // Player's level
JobAPI.getXp(uuid, "miner")            // Player's XP
JobAPI.getTotalLevel(uuid)             // Total level across all jobs
JobAPI.getTitle(uuid, "miner")         // Player's title
JobAPI.getSpecialization(uuid, "miner") // A, B, or null

Configuration

config.json

{
  "lang": "EN",
  "xpBoost": 1.0
}

jobs.json

Defines all 4 jobs with their XP sources, specializations, and tool IDs. Auto-generated on first run with default values.

quest_templates.json

Custom quest definitions per job. If empty, quests auto-generate from XP sources.

players.json

Player progress data (XP, levels, specializations, stats, quests). Auto-saved every 5 minutes.

All config files are stored in UserData/Saves/MODDING/mods/KatsuyaTV_PerfectJobs/

FAQ

Can players progress in multiple jobs at the same time?

Yes! All 4 jobs have independent progress. A player can be level 50 Miner, level 30 Lumberjack, etc.

Can I change the specialization after choosing?

No, specialization choice is permanent. Admins can reset a player's job via the admin panel.

How does the XP boost work?

The global XP multiplier (default 1.0x) is applied to all XP gains and quest rewards server-wide. Set it from the admin panel or config.

Does it work with PerfectEconomy?

Yes! If PerfectEconomy is installed, quest completions can award money rewards alongside XP.

How do tools upgrade?

Tools upgrade automatically when you reach the required level. Claim the new tier from the Jobs UI.
