[CENTER]
[SIZE=5][COLOR=#FF6B00][B]// WHY PERFECTSCOREBOARD?[/B][/COLOR][/SIZE]
[/CENTER]

[SIZE=3]Display a [B]beautiful HUD scoreboard[/B] on every player's screen. Fully configurable [B]title, lines, footer[/B] with [B]color codes[/B] and [B]dynamic placeholders[/B]. Multiple [B]profiles[/B] for different worlds, [B]animations[/B] (rainbow, scroll, flash), [B]custom logo[/B], and [B]5 screen positions[/B]. Everything managed through a clean [B]flat admin UI[/B] — no file editing needed.[/SIZE]

[HR][/HR]

[CENTER]
[SIZE=5][COLOR=#FF6B00][B]// FEATURES[/B][/COLOR][/SIZE]
[/CENTER]

[LIST]
[*][SIZE=4][COLOR=#55FF55][B]HUD Scoreboard[/B][/COLOR][/SIZE]
[SIZE=3]Displays automatically when players join. Up to [B]15 configurable lines[/B] with color codes (&a, &b, &c...) and dynamic placeholders. Semi-transparent background, clean design.[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Dynamic Placeholders[/B][/COLOR][/SIZE]
[SIZE=3]Built-in placeholders updated every second:
— [B]{player}[/B] — Player name
— [B]{online}[/B] — Online player count
— [B]{world}[/B] — Current world name
— [B]{time}[/B] — Server time (HH:mm)
— [B]{date}[/B] — Server date
— [B]{ping}[/B] — Player ping
— Plus [B]unlimited custom placeholders[/B] via command[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Multiple Profiles[/B][/COLOR][/SIZE]
[SIZE=3]Create different scoreboard layouts for different contexts:
— Assign profiles [B]per world[/B] (lobby, SMP, PvP each get their own scoreboard)
— Switch profiles on the fly via command or admin UI
— Each profile has its own title, lines, footer, animation, position, and logo setting[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Animations[/B][/COLOR][/SIZE]
[SIZE=3]4 animation modes per profile:
— [B]None[/B] — Static text
— [B]Rainbow[/B] — Color wave cycling through lines
— [B]Scroll[/B] — Scrolling text for long lines
— [B]Flash[/B] — Pulsing color effect[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]5 Screen Positions[/B][/COLOR][/SIZE]
[SIZE=3]Choose where the scoreboard appears:
— [B]Top-Right[/B] (default)
— [B]Top-Left[/B]
— [B]Bottom-Right[/B]
— [B]Bottom-Left[/B]
— [B]Center-Right[/B]
Configurable per profile via the admin UI.[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Custom Logo[/B][/COLOR][/SIZE]
[SIZE=3]Display your [B]server logo[/B] at the top of the scoreboard. Just place a PNG image in the plugin's texture folder. Toggle on/off per profile.[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Full Admin UI[/B][/COLOR][/SIZE]
[SIZE=3]Open [B]/sb admin[/B]:
— Profile list with create/select
— [B]Title & footer[/B] text fields
— [B]Line editor[/B] — click any line to edit, add or remove lines
— [B]Animation selector[/B] — cycle through modes
— [B]Logo toggle[/B] — ON/OFF
— [B]Position selector[/B] — cycle through 5 positions
— Logo file path displayed for easy setup[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Player Toggle[/B][/COLOR][/SIZE]
[SIZE=3]Players can [B]show/hide[/B] the scoreboard with [B]/sb[/B]. Preference persists across sessions.[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Custom Placeholders[/B][/COLOR][/SIZE]
[SIZE=3]Set custom global placeholders via [B]/sb setplaceholder kills 42[/B]. Use [B]{kills}[/B] in any line. Perfect for integrating with other plugins.[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Per-World Profiles[/B][/COLOR][/SIZE]
[SIZE=3]Assign different scoreboards to different worlds with [B]/sb setworld[/B]. Players see the right scoreboard automatically when they change worlds.[/SIZE]
[/LIST]
