[CENTER]
[SIZE=5][COLOR=#FF6B00][B]// WHY PERFECTKITS?[/B][/COLOR][/SIZE]
[/CENTER]

[SIZE=3]Give your players [B]starter kits[/B], [B]reward kits[/B], [B]rank kits[/B] — anything you want. Full [B]cooldown system[/B], [B]permission support[/B], [B]auto-give on first join[/B], and a clean [B]admin UI[/B] to manage everything without touching files.[/SIZE]

[HR][/HR]

[CENTER]
[SIZE=5][COLOR=#FF6B00][B]// FEATURES[/B][/COLOR][/SIZE]
[/CENTER]

[LIST]
[*][SIZE=4][COLOR=#55FF55][B]Kit Builder[/B][/COLOR][/SIZE]
[SIZE=3]Create kits via an [B]inventory picker UI[/B]. Add items, set quantities, customize display name and icon.[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Cooldowns[/B][/COLOR][/SIZE]
[SIZE=3]Per-player cooldown tracking. Set any duration in seconds per kit.[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Permissions[/B][/COLOR][/SIZE]
[SIZE=3]Optional permission requirement per kit. Control who can claim what.[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Auto-Give[/B][/COLOR][/SIZE]
[SIZE=3]Automatically give kits on [B]first join[/B]. Perfect for starter gear.[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Command Execution[/B][/COLOR][/SIZE]
[SIZE=3]Execute server commands when a kit is claimed. Run rewards, grant permissions, trigger events.[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Broadcast[/B][/COLOR][/SIZE]
[SIZE=3]Optional server-wide announcement when a player claims a kit.[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Admin UI[/B][/COLOR][/SIZE]
[SIZE=3]Full management panel via [B]/kit admin[/B]. Kit list, editor, item picker, all in one.[/SIZE]
[/LIST]
