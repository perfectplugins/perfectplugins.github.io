[CENTER]
[SIZE=5][COLOR=#FF6B00][B]// WHY PERFECTDEBUG?[/B][/COLOR][/SIZE]
[/CENTER]

[SIZE=3]Building Hytale plugins but can't find the API docs? PerfectDebug [B]automatically scans[/B] the server API, dumps every class, method, and field to text files, sniffs events, tracks vanilla assets, and monitors performance — all in [B]passive mode[/B]. Zero commands, zero config, zero impact. Just drop it in and get instant insight into Hytale's internals.[/SIZE]

[HR][/HR]

[CENTER]
[SIZE=5][COLOR=#FF6B00][B]// FEATURES[/B][/COLOR][/SIZE]
[/CENTER]

[LIST]
[*][SIZE=4][COLOR=#55FF55][B]API Scanner[/B][/COLOR][/SIZE]
[SIZE=3]Automatically scans [B]38+ server classes[/B] using reflection:
— Full method signatures with parameter types and return types
— All fields with types and modifiers
— Class hierarchy and interfaces
— Output: one text file per class in the plugin data folder
Covers: World, PlayerRef, Entity, ItemStack, Inventory, ItemContainer, ItemModule, HytaleServer, ComponentType, and more.
Scan completes in [B]~50ms[/B] at startup.[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Event Sniffer[/B][/COLOR][/SIZE]
[SIZE=3]Registers listeners on all available events and logs them:
— PlayerConnectEvent
— PlayerReadyEvent
— PlayerDisconnectEvent
— PlayerChatEvent
Output saved to [B]events.txt[/B].[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Vanilla Asset Scanner[/B][/COLOR][/SIZE]
[SIZE=3]Scans the Hytale asset directory and catalogs everything:
— [B]581 particles[/B] — All particle effect files
— [B]1,168 sounds[/B] — All sound effect files
— [B]356 textures[/B] — All block and item textures
Results saved to organized text files. Essential for finding the right particle/sound IDs for your mods.[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Performance Monitor[/B][/COLOR][/SIZE]
[SIZE=3]Periodic server health reports every [B]30 seconds[/B]. Track memory usage and performance trends.[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Passive Item Tracker[/B][/COLOR][/SIZE]
[SIZE=3]Logs items seen in player inventories. Useful for discovering item IDs and understanding what's available in the game.[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Player Tracker[/B][/COLOR][/SIZE]
[SIZE=3]Logs player connections and disconnections with timestamps.[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]100% Passive[/B][/COLOR][/SIZE]
[SIZE=3]No commands needed. No configuration. No server impact. Everything runs automatically at startup and outputs to files. Drop in, start server, check the output folder.[/SIZE]
[/LIST]
