[CENTER]
[SIZE=5][COLOR=#FF6B00][B]// WHY PERFECTWARPS?[/B][/COLOR][/SIZE]
[/CENTER]

[SIZE=3]The most complete warp system for Hytale. Create warps with [B]custom icons[/B], organize them by [B]categories[/B], let players mark [B]favorites[/B], and manage everything through a [B]clean flat UI[/B]. Movement cancellation, cooldowns, teleport stats — everything you need.[/SIZE]

[HR][/HR]

[CENTER]
[SIZE=5][COLOR=#FF6B00][B]// FEATURES[/B][/COLOR][/SIZE]
[/CENTER]

[LIST]
[*][SIZE=4][COLOR=#55FF55][B]Beautiful Warp Menu[/B][/COLOR][/SIZE]
[SIZE=3]Players open [B]/warps[/B] and see a clean list with [B]item icons[/B], descriptions, delay info, and a green [B]"Warp"[/B] button. Empty rows and pagination are hidden automatically.[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Categories & Tabs[/B][/COLOR][/SIZE]
[SIZE=3]Organize warps by category (Spawn, Shops, Arenas, etc.). Players filter by category using [B]tabs[/B] at the top of the menu.[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Favorites[/B][/COLOR][/SIZE]
[SIZE=3]Players mark warps as [B]favorites[/B] with a gold star. Filter by favorites to see only bookmarked warps. Persisted per player.[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Movement Cancellation[/B][/COLOR][/SIZE]
[SIZE=3]When a warp has a delay, the teleportation is [B]cancelled instantly if the player moves[/B]. Checked every 200ms — no cheating the countdown.[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Per-Player Cooldowns[/B][/COLOR][/SIZE]
[SIZE=3]Configurable cooldown per warp. Players see [B]remaining cooldown[/B] in the UI. Prevents spam.[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Teleport Stats[/B][/COLOR][/SIZE]
[SIZE=3]Each warp tracks [B]total teleportations[/B]. Visible in both player UI and admin panel.[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Full Admin UI[/B][/COLOR][/SIZE]
[SIZE=3]Open [B]/pw admin[/B]:
— Sidebar with paginated warp list
— [B]Text fields[/B] for name, description, icon, delay, permission, priority
— [B]Icon preview[/B] showing the selected item
— [B]"Set Here"[/B] to update position to your location
— [B]Confirmation before delete[/B] (click once = "Confirm?", click again = deleted)
— Create warps directly from the UI — no commands needed[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Import / Export[/B][/COLOR][/SIZE]
[SIZE=3][B]/pw export[/B] saves all warps to JSON. [B]/pw import[/B] loads them on another server. Easy migration.[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Max Warps Limit[/B][/COLOR][/SIZE]
[SIZE=3]Configurable maximum number of warps in [B]config.json[/B]. Set to 0 for unlimited.[/SIZE]
[*][SIZE=4][COLOR=#55FF55][B]Multi-language[/B][/COLOR][/SIZE]
[SIZE=3]All messages in [B]5 languages[/B]: EN, FR, ES, DE, IT.[/SIZE]
[/LIST]
